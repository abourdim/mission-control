Sample Lesson: Two-Device Rover Race

Ready-to-teach 45-minute example
Two-Device Rover Race: Driver & Spotter
45 minutes
Ages 9–14
STEM / Networking / Teamwork
Learning Objectives
1. Establish a peer-to-peer video call between two devices using a shared room code.
2. Send and interpret D-pad commands across a data channel as JSON.
3. Collaborate as a driver/spotter pair using remote speak and remote fullscreen.
4. (Optional) Bridge commands to a BBC micro:bit V2 and observe LED arrows mirror the D-pad.

Mini Mission Control features used in this lesson

  • PeerJS Video Call two-way camera + mic over a shared room code
  • D-pad Data Channel UP / DOWN / LEFT / RIGHT / STOP as pressed/release JSON
  • Remote Speak make the other device say text out loud with 🔊 / 📡
  • Logs Panel every TX/RX/ACK line visible to the whole team
  • micro:bit Bridge (optional) forwards commands to a BBC micro:bit V2 over BLE UART
