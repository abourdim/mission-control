Room-Code Rules
✅ Both devices must type the same room code.
Short words work: pizza, alpha-42, lab-7.
Same room code = same pairing channel.
One side becomes host, the other guest.
Change the code to make a fresh room.

Troubleshooting Quick Hits
Stuck on pairing? Check the room code matches exactly on both devices. Black self-view? Re-grant camera & mic permissions in Chrome. micro:bit won't pair? Needs HTTPS & a BBC micro:bit V2 with BLE UART firmware. Live connection required — this is not an offline app.
