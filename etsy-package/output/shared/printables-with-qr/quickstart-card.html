Mini Mission Control

Two-device peer-to-peer video call + D-pad + micro:bit bridge
Call • Command • Control • No install • No account
1
📱💻

Open on Two Devices

Load abourdim.github.io/mission-control/ in Chrome on both devices (laptop + phone works great). Camera & mic required; HTTPS for the optional micro:bit bridge.

2
🔤

Type the Same Room Code

Both devices must type the same room code. Anything short and memorable — alpha-42, pizza, your favorite word.

3
🎥

Start Camera

Press 🎥 Start camera and allow browser permissions. Your face appears in the self-view. Flip with camera switch or mic mute anytime.

4
✅

Connect

Press ✅ Connect. One device becomes host, the other guest. Video + audio stream peer-to-peer through PeerJS.

5
🎮

Enjoy — D-pad, Text, TTS, optional micro:bit link

Use the D-pad (UP/DOWN/LEFT/RIGHT/STOP) to send commands peer-to-peer. Type text to chat. Tap Speak for local TTS or Remote speak to make the other side talk. Optional: pair a BBC micro:bit V2 over BLE to forward D-pad commands to the board.
