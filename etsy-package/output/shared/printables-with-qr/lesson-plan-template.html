Sample Lesson: Pair, Command, Explore!

Ready-to-teach 45-minute example
Pair, Command, Explore! — A Peer-to-Peer Adventure
45 minutes
Ages 8–14
STEM / Digital Communication
Learning Objectives
1. Pair two devices using a shared room code.
2. Hold a peer-to-peer video + audio call.
3. Send D-pad commands and text messages across the channel.
4. Explain, in their own words, what "peer-to-peer" means.

Mission Control features used in this lesson

  • Room-Code Pairing both devices type the same code to find each other
  • P2P Video + Audio PeerJS direct connection — no central server
  • D-pad UP / DOWN / LEFT / RIGHT / STOP sent peer-to-peer
  • Text + TTS chat, local Speak, Remote speak over the data channel
  • Optional micro:bit bridge D-pad forwards to a BBC micro:bit V2 via BLE UART
