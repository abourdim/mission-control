Mini Mission Control — chatbot preview

This page is both a live preview of the pre-sale chatbot AND the embed snippet you paste into your live-demo page. The green button in the bottom-right is the chatbot — click it to try the canned FAQ rules.

To embed on your live demo: copy this snippet to the bottom of your demo page's HTML: 
<script>
// window.__bitChatBotApi = 'https://chatbot.your-domain.com/api/chat';
</script>
<script src="embed.js"></script>
Uncomment window.__bitChatBotApi + deploy api-stub-worker.js to enable LLM mode (rules-only is free).

What the chatbot answers

The canned rules in faq-rules.json cover the top pre-sale questions (compatibility, pairing, refunds, licensing, updates, privacy). Questions outside the rule set get a polite redirect to your Etsy inbox.

Files in this folder

  • embed.js — the widget (inject on any page)
  • faq-rules.json — edit these to change answers; redeploy with no code changes
  • api-stub-worker.js — optional Cloudflare Worker for LLM mode
  • embed.html — this preview page + copy-paste snippet
