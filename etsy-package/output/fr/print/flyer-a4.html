Vidéo pair-à-pair + D-pad prêts pour la classe. Sans serveur. Sans compte.

Démarrez en 60 secondes

  1. Ouvrez la page sur deux appareils.
  2. Tapez le même code de salle sur les deux.
  3. Pressez Démarrer la caméra — autorisez l'accès.
  4. Pressez Connecter — hôte + invité se relient.
  5. D-pad, texte, TTS, ou pont micro:bit.
