Mini Mission Control • Classroom Guide

1
💻📱

Open on TWO Devices

Open the same link — e.g. the GitHub Pages site — on a laptop and a phone. Chrome is best.

2
🔗

Match the Room Code

Type the same room code on both devices. Default: demo. Anything that matches works — 1234, mars, alpha…

3
🎥

Start Camera & Connect

Press 🎥 Start camera, allow permissions, then ✅ Connect. One device becomes host, the other guest — automatically.

4
🎮

Drive the D-pad

Hold ▲ UP, ▼ DOWN, ◀ LEFT, ▶ RIGHT to send commands. Tap ■ Stop to cut the motion.

5
🔊

Speak, Snap, Record

Type a message then tap 📡 Remote speak to make the other device say it. Capture a 📸 Snapshot or ⏺️ Record the remote video. Check the Logs panel if things feel haunted.

Two devices. One room code. Zero installs.

Optional: connect a BBC micro:bit V2 and flip Bridge mode on to steer a real robot from the other side of the room.
