Mini Mission Control

Two-device video call + D-pad command channel (laptop ↔ phone)
Call • Command • Control • No install, no account
1
🌐

Open on TWO Devices

Open the same link on a laptop and a phone — for example Chrome on both. GitHub Pages or any HTTPS host works.

2
🔗

Match the Room Code

In the Room box, type the same code on both devices (default: demo).

3
🎥

Start Camera

Press 🎥 Start camera on each device and allow camera & microphone when prompted.

4
✅

Connect

Press ✅ Connect. The first device becomes host; the second becomes guest. Remote video appears; status goes green.

5
🎮

Command & Speak

Hold the D-pad arrows to send UP / DOWN / LEFT / RIGHT / STOP. Type a message and tap Send, 🔊 Speak or 📡 Remote speak. Optional: connect a BBC micro:bit V2 over Bluetooth and flip Bridge mode on to forward commands to the board.
