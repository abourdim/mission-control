Etsy 60-sec Video Shoot Card Mini Mission Control · Script A

Pre-flight (90s)

  • Two devices ready (laptop + phone)
  • Chrome open on both, live URL loaded
  • Camera + mic permissions granted
  • Agreed room code picked (e.g. pizza)
  • Both devices typed the same room code
  • Cache cleaner run on both
  • Optional: BBC micro:bit V2 powered, BLE UART firmware ready
  • Phone in 9:16 on tripod, front-lit
  • Desk cleared — both devices + micro:bit visible
  • Captions app ready OR overlay plan
  • Script read once, under 55s
  • Phone silenced, kids/dogs warned

Specs (memorise)

Length≤ 60s (aim 55s)
Aspect9:16 portrait
Res1080 × 1920 min
CodecMP4 / H.264 + AAC
Size< 100 MB
AudioMuted by default → captions mandatory
CoverFirst frame = thumbnail
HookFirst 3s decides the scroll

During recording

  • One scene per phase — don't cut every 2s
  • Switch on words, not mid-sentence
  • 1s pause on Connect button
  • Eye contact with lens on CTA
  • Smile for final 3s — loop frame
  • If > 55s, re-cut

The 60-second arc — Script A

TimeSayDo / Click
0–3s "Two devices, one room code, and you're live — video, voice, and a D-pad." Split frame: phone + laptop both showing the Mission Control page.
3–10s "Type the same room code. Start camera. Connect." Type pizza on both. 🎥 Start camera. ✅ Connect. Remote video appears.
10–20s "Wave at the other side. Then press the D-pad." Wave on phone → laptop sees it. Tap ⬆ UP, ➡ RIGHT, ■ STOP. Log shows each arrive.
20–30s "Type a message — Enter. Remote speak makes it talk." Type "hello astronaut", Enter. Tap Remote speak → laptop speaker reads it.
30–40s "Pair a micro:bit V2 and the D-pad drives the board too." Pair micro:bit V2 over BLE. Tap D-pad arrows — board LEDs react.
40–50s "Snapshot. Record. Mirror. Fullscreen. All one tap." 📸 Snapshot → PNG downloads. ⏺ Record → WebM. Mirror toggle. Fullscreen remote.
50–55s "Link below — lifetime updates, teachers and kids welcome." Face-cam. Eye contact with lens.
55–60s "Welcome, tiny astronaut." Hold still. Logo + handle freeze. Loop frame.

After Stop

  • Watch once, eyes off script
  • Trim to 55s (iMovie / CapCut / DaVinci)
  • Burn in captions — centered below subject
  • Set first frame = strong still visual
  • Export MP4/H.264, < 100 MB
  • Upload · verify autoplay · phone + sound off
