Etsy 60-sec Video Shoot Card Mini Mission Control · Script A

Pre-flight (90s)

  • Two devices with cameras + mics (laptop + phone)
  • Chrome/Edge on both, page served over HTTPS
  • Same Wi-Fi; room code typed on both sides
  • Camera + mic permissions granted on each device
  • Language matches voiceover (EN / FR / AR)
  • Tagline visible on-screen overlay: Call • Command • Control
  • Optional micro:bit V2 paired via BLE (LED arrow demo)
  • Phone in 9:16 on tripod for split-screen framing
  • Desk cleared — laptop + phone + one prop only
  • Captions app ready OR overlay plan
  • Script read once, under 55s
  • Notifications silenced, kids/dogs warned

Specs (memorise)

Length≤ 60s (aim 55s)
Aspect9:16 portrait

CodecMP4 / H.264 + AAC
Size< 100 MB

CoverFirst frame = thumbnail
HookFirst 3s decides the scroll

During recording

  • One scene per phase — don't cut every 2s
  • Switch on words, not mid-sentence
  • Hold 1s on every D-pad press so CMD shows
  • Eye contact with lens on CTA
  • Land the tagline in the final 3s — loop frame
  • If > 55s, re-cut

The 60-second arc — Script A

TimeSayDo / Click
0–3s "Two devices. One room code. Call, Command, Control — in 60 seconds." Split-screen: laptop left, phone right, same page on both.
3–10s "Same room code on both sides — tap Start camera." Type matching room code. Tap Start camera. Video tiles light up green.
10–20s "Hit Connect — live video, live audio, peer to peer." Click Connect. Two faces side by side. HUD overlay shows peer status.
20–30s "Kid taps UP on the D-pad — laptop receives the command instantly." Phone: press UP on D-pad. Laptop shows CMD UP. Optional micro:bit V2 lights arrow.
30–40s "Type a message, press Speak — it reads aloud on the other device." Type → Send. Tap Speak. Remote device talks back.
40–50s "Snapshot the moment, record the call, done — no install, no account." Snapshot thumbnail saves. Toggle Record. Quick cut HUD ↔ tiles.
50–55s "Link below — lifetime updates, works on any two devices with cameras." Face-cam. Eye contact with lens.
55–60s "Call. Command. Control." Hold still. Tagline + logo freeze. Loop frame.

After Stop

  • Watch once, eyes off script
  • Trim to 55s (iMovie / CapCut / DaVinci)
  • Burn in captions — centered below subject
  • Set first frame = strong still visual (split-screen)
  • Export MP4/H.264, < 100 MB
  • Upload · verify autoplay · phone + sound off
