الوصف الكامل

🎮 Mini Mission Control — مكالمة فيديو + قناة أوامر D-pad بين جهازين، كلها في المتصفح. افتح ملف HTML واحداً على حاسوب وهاتف، اكتب نفس رمز الغرفة، وستحصل على: فيديو وصوت ثنائي الاتجاه، D-pad ترسل الأوامر كـ JSON، رسائل نصية، وتحويل النص إلى كلام — بدون تثبيت، بدون حساب، بدون خادم، بدون تتبُّع. ما ستحصل عليه • 📹 مكالمة فيديو + صوت ثنائية الاتجاه (PeerJS / WebRTC) • 🎮 قناة أوامر D-pad — UP / DOWN / LEFT / RIGHT / STOP مع أحداث الضغط والإفلات • ✉️ رسائل نصية على قناة البيانات • 🔊 Speak محلي و📡 Remote Speak (جعل الجهاز الآخر يتحدث) • 📸 لقطة PNG و⏺️ تسجيل WebM للبث البعيد • ⛶ طلب ملء الشاشة البعيد و👁 طبقة HUD على الفيديو • 🪞 مرآة ذاتية، ↔️ قلب بعيد، 🔇 كتم الميكروفون، 🔄 تبديل الكاميرا • 🧩 جسر اختياري إلى BBC micro:bit V2 عبر BLE UART (يرسل CMD UP 1 / CMD STOP 1) • 📝 لوحة Logs تعرض كل حزمة TX / RX / ACK محتوى الملف • index.html — لوحة تحكم المهمة • js/app.js — PeerJS، الكاميرا، D-pad، جسر BLE • css/style.css — مظهر وحدة تحكم داكن • makecode.ts — برنامج اختياري لـ BBC micro:bit V2 • README.md + USERGUIDE.html — دليل كامل • etsy-package/ — مطبوعات: بطاقة البداية السريعة، ورقة اختصارات، ملصق A3، ملصقات، خطة درس • LICENSE.txt — ترخيص فردي شخصي / صفّي المتطلبات • جهازان لكل منهما كاميرا وميكروفون (حاسوب + هاتف هو الأنسب) • Chrome مُستحسَن على كليهما (Safari: فيديو نعم، Bluetooth لا) • استضافة HTTPS (GitHub Pages ممتاز) أو localhost • شبكة تسمح بالاتصال المباشر (بعض شبكات المدارس تحجبه) • اختياري: BBC micro:bit V2 لجسر Bluetooth لمن هو هذا المنتج المعلّمون في ورش STEM · عائلات التعليم المنزلي · نوادي الروبوتات · المبتكرون الذين يبنون تطبيقات تحكُّم عن بُعد · كل من يريد قيادة شيء من غرفة أخرى. الخصوصية أولاً الفيديو والصوت والأوامر تنتقل مباشرةً بين الجهازين. بدون حسابات، بدون تحليلات، بدون تخزين سحابي للتسجيلات. كل شيء يبقى في متصفحك. تحديثات مدى الحياة كل نسخة تُطرَح عبر هذا الإعلان تصبح ملكك إلى الأبد. أعد تحميل آخر ZIP في أي وقت من صفحة مشترياتك على Etsy. قيادة سعيدة! 🚀🎮🧑‍🚀

ترتيب الصور السبع

تسلسل الصور (Etsy يعرض الأولى والثانية بالحجم الأكبر)

  1. الصورة البارزة — حاسوب + هاتف جنباً إلى جنب، كلاهما يعرض اللوحة وشارات الحالة خضراء. عنوان بخط Orbitron.
  2. رمز الغرفة المشترك — لقطة قريبة لخانتَي Room تعرضان نفس الرمز، مع سهم بينهما.
  3. D-pad عن قرب — شاشة الهاتف مع D-pad مضغوطة، ولوحة Logs على الحاسوب تطبع سطور TX / RX.
  4. Remote speak — جهاز يكتب، والآخر مع موجة صوتية.
  5. الجسر الاختياري — هاتف وحاسوب وBBC micro:bit V2 يعرض سهم LED يطابق ضغطة D-pad.
  6. المطبوعات — بطاقة البداية السريعة، ورقة الاختصارات، ملصق A3، ملصقات في صورة flat-lay.
  7. بطاقة نصية — «ترخيص لمستخدم واحد · تحديثات مدى الحياة · بلا سحابة · Chrome مُستحسَن» على تدرّج داكن.
