SLUG — SCENE-BY-SCENE SCRIPT

6-scene script — Mini Mission Control

Call · Command · Control — peer-to-peer video + D-pad + walkie-talkie in one HTML file. Shoot in this exact order.

00:00–00:03HOOK
🎙 SAY
"Two devices. One room code. Instant D-pad across the world."
🎬 DO
Split frame: phone on the left, laptop on the right. Both have the app open. Same 4-letter room code (e.g. ASTRO) highlighted on each screen.
00:03–00:10CONTEXT
🎙 SAY
"No server. No account. Just a code both devices type in."
🎬 DO
Type ASTRO on both devices. Paired status chip flips to green. Both camera previews pop in — peer-to-peer WebRTC via PeerJS, no cloud.
00:10–00:25DEMO 1 · VIDEO + D-PAD
🎙 SAY
"Video call live — and every button press jumps across instantly."
🎬 DO
Both devices show each other's camera. A kid presses UP on the phone → the laptop's D-pad lights UP in real time and the RX bar prints UP pressed. Press LEFT, RIGHT, STOP — each one mirrors instantly.
00:25–00:40DEMO 2 · TEXT + REMOTE TTS
🎙 SAY
"Type here, speak there. Walkie-talkie for real families."
🎬 DO
Parent types "Dinner ready!" on phone, taps REMOTE SPEAK — the laptop's speakers say the words out loud through the browser TTS. Show the RX log scrolling on the receiving device.
00:40–00:50PAYOFF · micro:bit bridge
🎙 SAY
"FaceTime + GameController + walkie-talkie. In one HTML file."
🎬 DO
(Optional hardware beat.) A BBC micro:bit V2 sits between the two devices. Press UP on the phone D-pad — the micro:bit's LED matrix draws the UP arrow. Press LEFT — arrow rotates. D-pad commands flow over BLE UART to the board.
00:50–00:60CTA + END FRAME
🎙 SAY
"Peer-to-peer. Chrome or Edge. Kid-safe. Link below."
🎬 DO
End on the split-frame shot with both cameras live and the green paired chip. Overlay the shop handle and Mini Mission Control logo. 3-second hold so the loop starts on this frame.

🧑‍🚀 Mini Mission Control — Call · Command · Control. Two devices. One room code. One HTML file. Instant peer-to-peer video call + cross-device D-pad remote + text-to-speech walkie-talkie — all in the browser, no server, no account, no cloud. WHAT YOU GET • 📹 Two-device peer-to-peer video + audio call via PeerJS (WebRTC) — laptop↔phone, tablet↔laptop, two phones, any combo • 🎮 Shared 5-button D-pad (UP / DOWN / LEFT / RIGHT / STOP) — press on device A, fires on device B in real time • 💬 Text channel with Local TTS ("Speak" — say it here) and Remote TTS ("Remote speak" — make the other device speak out loud) • 🔑 Room-code pairing — type the same 4-letter code on both devices, they link. No accounts, no logins. • 🤖 Optional BBC micro:bit V2 bridge — D-pad commands forward over BLE UART and light up arrow icons on the LED matrix • 🎥 Camera tools — switch front/back, mute mic, mirror self, mirror remote, request remote fullscreen • 📸 Capture tools — PNG snapshot, WebM recording, HUD overlay toggle, live RX command bar • 📜 Log panel — TX + RX + system events, local-only, cache cleaner built in • 🧑‍🚀 Astronaut-themed onboarding: "Welcome, tiny astronaut" WHAT'S IN YOUR DOWNLOAD • index.html — the full app (video + D-pad + TTS, no build step) • makecode.ts — optional micro:bit V2 firmware (flash once from MakeCode) • SETUP.md — 5-minute quick-start • docs/ — illustrated guide, cheat sheet, FAQ, Etsy playbook • LICENSE — personal / classroom single-user license REQUIREMENTS • Two devices with Chrome or Edge (laptop + phone, tablet + laptop, or two phones) • Camera + microphone permission allowed in the browser • Optional: BBC micro:bit V2 for the BLE LED-arrow bridge (HTTPS required for Bluetooth) • Works offline over file:// for video/D-pad/TTS; use HTTPS when you want the micro:bit bridge WHO IT'S FOR Families who want a kid-safe walkie-talkie between rooms · teachers running remote STEM demos · robotics clubs pairing a controller to a robot · makers prototyping peer-to-peer WebRTC projects · curious tiny astronauts 🧑‍🚀 PRIVACY FIRST Peer-to-peer only. Video and audio stream device-to-device over WebRTC — no analytics, no logins, no cloud recording. The only outside traffic is a short pairing handshake through the public PeerJS signaling broker. Snapshots, recordings, and logs live on your disk. LIFETIME UPDATES Every version shipped from this listing is yours forever. Re-download the latest ZIP anytime from your Etsy Purchases page. Common mistakes
