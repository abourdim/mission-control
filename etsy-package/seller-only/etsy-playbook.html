Full description

🎮 Mini Mission Control — a browser-based video call + command channel between two devices. Open one HTML file on a laptop and a phone, match a room code, and you get live two-way video, a D-pad that streams commands as JSON, text messaging, and text-to-speech — no install, no account, no backend, no tracking. WHAT YOU GET • 📹 Two-way video + audio call (PeerJS / WebRTC) • 🎮 D-pad command channel — UP / DOWN / LEFT / RIGHT / STOP with pressed/release events • ✉️ Text messaging over the data channel • 🔊 Local Speak and 📡 Remote Speak (make the other device talk) • 📸 Snapshot (PNG) and ⏺️ Record (WebM) the remote feed • ⛶ Remote fullscreen request and 👁 HUD overlay on the video • 🪞 Mirror self, ↔️ Flip remote, 🔇 Mute mic, 🔄 Switch front/rear camera • 🧩 Optional BBC micro:bit V2 bridge over BLE UART (forwards commands as CMD UP 1 / CMD STOP 1 lines) • 📝 Logs panel showing every TX / RX / ACK packet WHAT'S IN YOUR DOWNLOAD • index.html — the mission console • js/app.js — PeerJS, camera, D-pad, BLE bridge • css/style.css — dark console look • makecode.ts — optional firmware for a BBC micro:bit V2 • README.md + USERGUIDE.html — full walkthrough • etsy-package/ — printables: quick-start card, reference sheet, A3 poster, sticker sheet, lesson plan • LICENSE.txt — personal / classroom single-user license REQUIREMENTS • Two devices with cameras + microphones (laptop + phone is typical) • Chrome recommended on both (Safari works for video, not for Bluetooth) • HTTPS host (GitHub Pages works great) or localhost • A network that allows peer-to-peer (some school Wi-Fi blocks it) • Optional: a BBC micro:bit V2 for the Bluetooth bridge WHO IT'S FOR Teachers running STEM workshops · home-schooling families · robotics clubs · makers prototyping remote-control projects · anyone who wants to drive a thing from another room. PRIVACY FIRST Video, audio, and commands go peer-to-peer between the two devices. No accounts, no analytics, no cloud storage of recordings. Everything stays in your browser. LIFETIME UPDATES Every version shipped from this listing is yours forever. Re-download the latest ZIP anytime from your Etsy Purchases page. Happy driving! 🚀🎮🧑‍🚀
