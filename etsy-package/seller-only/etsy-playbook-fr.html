Description complète

🎮 Mini Mission Control — un appel vidéo entre deux appareils avec canal de commandes D-pad, dans le navigateur. Ouvrez un seul fichier HTML sur un ordinateur et un téléphone, saisissez le même code de salle, et vous obtenez : vidéo + audio bidirectionnels, une D-pad qui envoie des commandes JSON, des messages texte et la synthèse vocale — sans installation, sans compte, sans serveur, sans tracking. CE QUE VOUS OBTENEZ • 📹 Appel vidéo + audio bidirectionnel (PeerJS / WebRTC) • 🎮 Canal de commandes D-pad — UP / DOWN / LEFT / RIGHT / STOP avec événements appui / relâchement • ✉️ Messagerie texte sur le canal de données • 🔊 Speak local et 📡 Remote Speak (faire parler l’autre appareil) • 📸 Snapshot PNG et ⏺️ Enregistrement WebM du flux distant • ⛶ Demande de plein écran distant et 👁 incrustation HUD sur la vidéo • 🪞 Miroir local, ↔️ Flip distant, 🔇 Mute micro, 🔄 Caméra avant / arrière • 🧩 Pont optionnel vers une BBC micro:bit V2 en BLE UART (transmet CMD UP 1 / CMD STOP 1) • 📝 Panneau Logs qui affiche chaque paquet TX / RX / ACK CONTENU DU TÉLÉCHARGEMENT • index.html — la console de mission • js/app.js — PeerJS, caméra, D-pad, pont BLE • css/style.css — look console sombre • makecode.ts — firmware optionnel pour la BBC micro:bit V2 • README.md + USERGUIDE.html — guide complet • etsy-package/ — imprimables : carte de démarrage, aide-mémoire, affiche A3, stickers, plan de leçon • LICENSE.txt — licence mono-utilisateur personnelle / classe PRÉREQUIS • Deux appareils avec caméra et micro (ordinateur + téléphone, typiquement) • Chrome recommandé sur les deux (Safari : vidéo OK, Bluetooth non) • HTTPS (GitHub Pages fonctionne bien) ou localhost • Un réseau qui autorise le pair-à-pair (certains Wi-Fi scolaires le bloquent) • Optionnel : une BBC micro:bit V2 pour le pont Bluetooth POUR QUI Enseignants en ateliers STEM · familles en école à la maison · clubs de robotique · makers qui prototypent du pilotage à distance · toute personne curieuse de piloter un objet d’une autre pièce. CONFIDENTIALITÉ D’ABORD Vidéo, audio et commandes circulent directement entre les deux appareils en pair-à-pair. Pas de comptes, pas d’analytique, aucun enregistrement envoyé dans le cloud. Tout reste dans votre navigateur. MISES À JOUR À VIE Chaque version publiée depuis cette annonce vous appartient. Retéléchargez le dernier ZIP à tout moment depuis votre page Achats Etsy.
