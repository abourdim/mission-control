Description complète

🎮 Mini Mission Control — Appeler • Commander • Contrôler. Ouvrez la page sur deux appareils. Tapez le même code de room. Pressez Connecter. Vous avez un appel vidéo pair-à-pair avec un D-pad entre les deux — dans le navigateur, sans installation, sans compte. Pensez : FaceTime + manette de jeu + talkie-walkie micro:bit. CE QUE VOUS OBTENEZ • 📞 Appel vidéo + audio pair-à-pair entre deux appareils (PeerJS) • 🔤 Appairage par code de room (les deux appareils tapent le même code) • 🎮 D-pad : HAUT / BAS / GAUCHE / DROITE / STOP envoyés en pair-à-pair • 💬 Messages texte via le canal de données • 🗣️ Synthèse vocale locale (Speak) + Remote speak (faire parler l'autre appareil) • 📡 Pont BBC micro:bit V2 optionnel — le D-pad est transmis à la carte via BLE UART • 🔄 Changement de caméra, 🎙️ coupure micro, 🪞 miroir soi / distant, ⛶ plein écran distant • 📸 Capture (PNG), ⏺ enregistrement (WebM) • 🛰️ Superposition HUD, 📋 panneau de log, 🧹 nettoyeur de cache CONFIGURATION EN 5 ÉTAPES 1. Ouvrez la page sur deux appareils 2. Tapez le même code de room sur les deux 3. Pressez 🎥 Démarrer la caméra (autorisez les permissions) 4. Pressez ✅ Connecter (l'un devient hôte, l'autre invité) 5. Profitez — D-pad, texte, TTS, pont micro:bit optionnel CONTENU DU TÉLÉCHARGEMENT • index.html — l'application complète • js/ — câblage PeerJS, D-pad, TTS, pont micro:bit optionnel • docs/ — guide illustré, référence rapide, FAQ, playbook Etsy • Extras imprimables : carte de démarrage, poster A3, plan de leçon, autocollants • LICENSE — licence mono-utilisateur personnelle / classe PRÉREQUIS • Deux appareils avec navigateur (Chrome recommandé) • Caméra + microphone sur chaque appareil • Une connexion réseau active (c'est un appel pair-à-pair, pas hors ligne) • Optionnel : BBC micro:bit V2 pour le pont BLE • HTTPS obligatoire pour Bluetooth RACCOURCIS CLAVIER Seulement deux : Entrée (envoyer le texte), Échap (fermer le modal firmware). POUR QUI Enseignants en ateliers STEM / communication numérique · familles en école à la maison · clubs de robotique · makers prototypant des projets P2P · toute personne curieuse du web pair-à-pair. CONFIDENTIALITÉ D'ABORD La vidéo et l'audio circulent directement entre les deux appareils. Un minuscule service de signalisation les aide seulement à se trouver — le contenu de l'appel reste pair-à-pair.
