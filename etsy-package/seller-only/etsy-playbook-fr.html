Description complète

🧑‍🚀 Mini Mission Control — Call · Command · Control. Deux appareils. Un code de salle. Un seul fichier HTML. Appel vidéo pair-à-pair + D-pad partagé entre appareils + talkie-walkie à synthèse vocale — directement dans le navigateur, sans serveur, sans compte, sans cloud. CE QUE VOUS OBTENEZ • 📹 Appel vidéo + audio pair-à-pair entre deux appareils via PeerJS (WebRTC) — laptop↔téléphone, tablette↔laptop, deux téléphones, toutes combinaisons • 🎮 D-pad partagé à 5 boutons (UP / DOWN / LEFT / RIGHT / STOP) — appuyez sur l'appareil A, ça se déclenche sur l'appareil B en temps réel • 💬 Canal texte avec TTS local (« Speak » — lire ici) et TTS distant (« Remote speak » — faire parler l'autre appareil à voix haute) • 🔑 Appairage par code de salle — tapez le même code à 4 lettres sur les deux appareils, ils se connectent. Aucun compte, aucune connexion. • 🤖 Pont optionnel BBC micro:bit V2 — les commandes du D-pad traversent en BLE UART et allument les flèches sur la matrice LED • 🎥 Outils caméra — bascule avant/arrière, coupure micro, miroir soi, miroir distant, demande de plein écran distant • 📸 Outils capture — snapshot PNG, enregistrement WebM, overlay HUD, barre RX des commandes en direct • 📜 Panneau de logs — TX + RX + événements système, 100% local, nettoyeur de cache intégré • 🧑‍🚀 Onboarding thème astronaute : « Bienvenue, petit astronaute » CONTENU DU TÉLÉCHARGEMENT • index.html — l'application complète (vidéo + D-pad + TTS, sans build) • makecode.ts — firmware micro:bit V2 optionnel (à flasher une fois depuis MakeCode) • SETUP.md — démarrage rapide en 5 minutes • docs/ — guide illustré, aide-mémoire, FAQ, playbook Etsy • LICENSE — licence mono-utilisateur personnelle / classe PRÉREQUIS • Deux appareils avec Chrome ou Edge (laptop + téléphone, tablette + laptop, ou deux téléphones) • Autorisations caméra + micro accordées dans le navigateur • Optionnel : BBC micro:bit V2 pour le pont LED-flèche en BLE (HTTPS requis pour Bluetooth) • Fonctionne hors ligne en file:// pour vidéo/D-pad/TTS ; utilisez HTTPS pour activer le pont micro:bit POUR QUI Familles qui veulent un talkie-walkie sans danger entre les pièces · enseignants qui animent des démos STEM à distance · clubs robotiques qui appairent une manette à un robot · makers qui prototypent du WebRTC pair-à-pair · petits astronautes curieux 🧑‍🚀 CONFIDENTIALITÉ D'ABORD Pair-à-pair uniquement. Vidéo et audio circulent d'appareil à appareil en WebRTC — aucune analytique, aucune connexion, aucun enregistrement cloud. Le seul trafic externe est une courte poignée de main d'appairage via le broker public PeerJS. Captures, enregistrements et logs restent sur votre disque. MISES À JOUR À VIE Chaque version publiée depuis cette annonce est à vous pour toujours. Retéléchargez le dernier ZIP à tout moment depuis votre page Achats Etsy. Bon hacking ! 🚀🎮🤖
