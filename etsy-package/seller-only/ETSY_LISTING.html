3. Full description

🎮 Mini Mission Control — a browser-based video call + command channel between two devices.

Open one HTML file on a laptop and a phone, match a room code, and you get live two-way video, a D-pad that streams commands as JSON, text messaging, and text-to-speech — no install, no account, no backend, no tracking.

WHAT YOU GET

  • 📹 Two-way video + audio call (PeerJS / WebRTC under the hood)
  • 🎮 D-pad command channel — UP / DOWN / LEFT / RIGHT / STOP with pressed/release events as JSON
  • ✉️ Text messaging over the data channel
  • 🔊 Local Speak and 📡 Remote Speak (make the other device talk)
  • 📸 Snapshot (PNG) and ⏺️ Record (WebM) the remote feed
  • ⛶ Remote fullscreen request and 👁 HUD overlay on the video
  • 🪞 Mirror self, ↔️ Flip remote, 🔇 Mute mic, 🔄 Switch front/rear camera
  • 🧩 Optional BBC micro:bit V2 bridge over BLE UART (forwards commands as CMD UP 1 / CMD STOP 1 lines)
  • 📝 Logs panel showing every TX / RX / ACK packet

WHAT'S IN YOUR DOWNLOAD

  • index.html — the mission console
  • js/app.js — PeerJS, camera, D-pad, BLE bridge
  • css/style.css — dark console look
  • makecode.ts — optional firmware for a BBC micro:bit V2
  • README.md + USERGUIDE.html — full walkthrough
  • etsy-package/ — printables: A4 quick-start, quick-reference sheet, A3 classroom poster, 30 achievement stickers, 45-minute lesson plan
  • LICENSE.txt — personal / classroom single-user license

REQUIREMENTS

  • Two devices with cameras + microphones (laptop + phone is typical)
  • Chrome recommended on both (Safari works for video, not for the Bluetooth bridge)
  • HTTPS host (GitHub Pages works great) or localhost
  • A network that allows peer-to-peer (some school Wi-Fi blocks it)
  • Optional: a BBC micro:bit V2 for the Bluetooth bridge

WHO IT'S FOR

Teachers running STEM workshops · home-schooling families · robotics clubs · makers prototyping remote-control projects · anyone who wants to drive a thing from another room.

PRIVACY FIRST

Video, audio, and commands go peer-to-peer between the two devices. No accounts, no analytics, no cloud storage of recordings. Everything stays in your browser.

LIFETIME UPDATES

Every version shipped from this listing is yours forever. Re-download the latest ZIP anytime from your Etsy Purchases page.

11. FAQ snippets

Q: Do I need a micro:bit to use this?
No. The video call + D-pad + remote speak all work with just two browsers. The micro:bit bridge is an optional bonus.

Q: Does it work on iPhone / Safari?
Video and audio work on Safari. The optional Bluetooth bridge requires Chrome. For the full experience, use Chrome on both.

Q: Does it work on school Wi-Fi?
Usually yes, but some networks block peer-to-peer. Test first with a phone hotspot as a backup.

Q: Is any video recorded or uploaded?
No. The connection is peer-to-peer. Snapshots/recordings save locally to the capturing device only.

Q: Can I use it across my school or district?
Yes. Site license covers up to 30 teachers at one school; district license covers multiple sites.
