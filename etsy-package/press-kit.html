Short paragraph

Long paragraph

Mini Mission Control is a peer-to-peer video + D-pad commands app for the BBC micro:bit V2, built on WebRTC. Classic classroom remote-control demos require a server, an account, and a cloud relay — all of which fail school privacy reviews. Mini Mission Control flattens that journey: open the app on two devices, share a room link or QR code, and video streams direct between them via WebRTC. No cloud server touches the stream. A D-pad overlay on either side sends directional commands to the peer. Pair a BBC micro:bit V2 on the "controller" device and use its physical buttons plus accelerometer as the D-pad input — the commands travel over the same WebRTC data channel as the video. Fully client-side — no cloud, no account, no telemetry, COPPA/GDPR-friendly from day one. Trilingual UI (EN/FR/AR with full RTL), keyboard shortcuts, QR-based pairing for tablets, and a simulator mode for teachers testing without hardware. One of the few peer-to-peer remote-control kits on Etsy that keeps video off the cloud entirely.
