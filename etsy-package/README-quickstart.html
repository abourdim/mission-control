Mini Mission Control

Two-Device Video Call + D-pad Command Channel (laptop ↔ phone)
v1.0.0 • Digital Download
Thank you for buying Mini Mission Control! You just grabbed a zero-install, zero-account, browser-local mission console that pairs two devices over a shared room code — live video, live audio, a D-pad command channel, text-to-speech, and an optional Bluetooth bridge to a BBC micro:bit V2. Everything runs in your browser. No cloud, no telemetry, no accounts.

First 3 Things To Do

1
Open the site on TWO devices. Serve index.html from GitHub Pages (HTTPS) or run python -m http.server 8000 locally. Open the URL on both a laptop and a phone, in Chrome.
2
Match the room code. Type the same text (default: demo) in the Room box on both devices. Press 🎥 Start camera, allow permissions, then ✅ Connect. One device auto-picks host, the other guest.
3
Play. Hold the D-pad arrows to send UP/DOWN/LEFT/RIGHT. Tap 📡 Remote speak to make the other device talk. Optional: connect a BBC micro:bit V2 over BLE and flip Bridge mode on.

Requirements

✓Two devices (laptop + phone works great) — each with camera + microphone
✓Chrome is best on both. Safari/iPhone works for video but not for Bluetooth.
✓HTTPS host (GitHub Pages) or localhost for camera permissions
✓A Wi-Fi network that permits peer-to-peer (some school Wi-Fi blocks it)
✓Optional: a BBC micro:bit V2 for the Bluetooth bridge

Key Features

Call: two-way video + audio (PeerJS / WebRTC) • mute mic • switch camera • mirror / flip • hang up

Command: D-pad (UP / DOWN / LEFT / RIGHT / STOP) sent as pressed/release JSON over the data channel

Control: text messages • local speak • remote speak • snapshot (PNG) • record (WebM) • HUD overlay • remote fullscreen request

Bridge (optional): forwards D-pad commands to a BBC micro:bit V2 over BLE UART as CMD UP 1 / CMD STOP 1 lines

Privacy

100% Peer-to-Peer — No Account, No Cloud Storage

Video, audio, and commands stream directly between your two devices via PeerJS's signalling. No recordings are sent anywhere by default. Everything stays local. Safe for classroom use.
